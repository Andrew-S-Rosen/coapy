# coapy

Welcome to the documentation for `coapy`, a Python package for generating a list of collaborators suitable for an [NSF Collaborators and Other Affiliations (COA) document](https://www.nsf.gov/bfa/dias/policy/coa/coa_template.xlsx) or [DOE Collaborators form](https://science.osti.gov/-/media/grants/excel/Collaborator_Template.xlsx).
